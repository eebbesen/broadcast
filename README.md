# Broadcast
Send SMS to distinct groups of recipients


# Developers
## Prerequisities
* Have a PostgreSQL instance available

## Setup
```bash
bundle install
bin/rails db:create
```

## Tests
```bash
rspec
```

## Linters
```
bundle exec rubocop
bundle exec brakeman
bundle exec bundler-audit
bundle exec reek
```
